# ATM System (Python)

## 📌 Project Description
This is a console-based ATM simulation system developed in Python.  
The program allows users to create accounts, securely log in using a PIN, and perform banking operations such as deposits, withdrawals, balance checks, and viewing transaction history.

---

## 🚀 Features

- Create a new bank account
- Secure login with PIN
- Check account balance
- Deposit money
- Withdraw money
- View transaction history
- Logout system
- Menu-driven interface

---

## 🛠 Technologies Used

- Python 3
- Dictionary data structure
- Functions
- Loops and conditionals

---

## 📂 How the System Works

Each account stores:
- Account Number
- Name
- PIN
- Balance
- Transaction History

All account data is stored temporarily using a Python dictionary.

---

## ▶️ How to Run the Program

1. Make sure Python is installed on your system.
2. Download the file:
   atm_system.py
3. Open terminal or command prompt.
4. Run the program:

   python atm_system.py

---

## ⚠️ Note

- This project stores data temporarily (no database used).
- All data will be lost after the program stops.
- This is a basic simulation project for learning purposes.
